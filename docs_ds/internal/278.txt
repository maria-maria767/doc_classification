Critères de qualité:

* Le site web doit être facile à utiliser et à naviguer pour les utilisateurs finaux.
* Le site web doit être optimisé pour les moteurs de recherche (SEO).
* Le site web doit être compatible avec les différents navigateurs web (Chrome, Firefox, Safari, etc.).
* Le site web doit être sécurisé et protégé contre les attaques externes.

Moyens:

* Outils de conception: Adobe XD, Sketch, Figma
* Outils de développement: HTML, CSS, JavaScript, PHP, MySQL
* Outils de déploiement: Git, GitHub, Netlify
